# PageRankAI - Public Assets

This directory contains public assets for the PageRankAI Shopify app.

## Directory Structure

```
public/
├── images/          # Image assets (logos, icons, illustrations)
├── fonts/           # Custom web fonts
└── favicon.ico      # Site favicon
```

## Usage

Files in this directory are served statically and can be referenced in your code as:

```jsx
<img src="/images/logo.png" alt="PageRankAI Logo" />
```

## Asset Guidelines

- **Images**: Use optimized formats (WebP, optimized PNG/JPG)
- **Logos**: Provide multiple sizes for different use cases
- **Icons**: Consider using SVG for scalability
- **Fonts**: Only include necessary font weights to reduce load time
