# System Prompts

This directory contains system-level prompts for AI interactions used throughout the PageRankAI Shopify app.

## Files Structure

- `seo-analysis.txt` - Prompts for analyzing product page SEO
- `content-optimization.txt` - Prompts for optimizing product descriptions
- `keyword-generation.txt` - Prompts for generating relevant keywords

## Usage

These prompts should be used by the AI service to maintain consistency in AI-generated responses and analysis.
