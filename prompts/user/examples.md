# Example Prompts and Use Cases

This document provides practical examples of how to use PageRankAI's AI-powered features.

## Example 1: Basic SEO Analysis

**Scenario:** You have a new product page that's not getting traffic.

**Prompt:**
```
Analyze my product page at https://mystore.com/products/organic-cotton-tshirt

Target keywords: organic cotton t-shirt, sustainable fashion, eco-friendly clothing
Current ranking: Not in top 100
Goal: Rank in top 10 for "organic cotton t-shirt"
```

**Expected Output:**
- SEO score and analysis
- Specific optimization recommendations
- Keyword suggestions
- Content improvements

---

## Example 2: Content Optimization

**Scenario:** Rewriting a product description to be more SEO-friendly.

**Prompt:**
```
Optimize this product description:

Current: "This is a nice t-shirt made from cotton. Available in multiple colors."

Product: Premium Organic Cotton T-Shirt
Target keyword: organic cotton t-shirt
Brand voice: Eco-conscious, premium quality
Unique feature: 100% GOTS certified organic cotton
```

**Expected Output:**
- SEO-optimized description
- Keyword integration
- Compelling copy
- Call-to-action

---

## Example 3: Keyword Research

**Scenario:** Finding the best keywords for a new product.

**Prompt:**
```
Generate keywords for:

Product: Handmade ceramic coffee mug
Category: Home & Kitchen
Target audience: Coffee enthusiasts, age 25-45
Budget: Mid-range ($25-40)
Unique features: Microwave safe, artisan crafted, minimalist design
```

**Expected Output:**
- Primary keyword suggestions
- Long-tail variations
- Search volume estimates
- Competition analysis

---

## Example 4: Competitor Analysis

**Scenario:** Understanding how to outrank competitors.

**Prompt:**
```
Compare my product page with competitors:

My page: https://mystore.com/products/yoga-mat
Competitor 1: https://competitor1.com/premium-yoga-mat
Competitor 2: https://competitor2.com/yoga-mat-pro

Focus on: Content quality, keyword usage, page structure
```

**Expected Output:**
- Gap analysis
- Competitive advantages
- Opportunities to differentiate
- Strategic recommendations

---

## Example 5: Bulk Optimization

**Scenario:** Optimizing multiple products in a category.

**Prompt:**
```
Provide SEO strategy for my jewelry collection:

Category: Handmade Silver Jewelry
Number of products: 25
Common keywords: handmade jewelry, silver jewelry, artisan jewelry
Current average ranking: 50-80
Goal: Improve to top 20
```

**Expected Output:**
- Category-level strategy
- Product-specific recommendations
- Internal linking structure
- Content plan

---

## Tips for Effective Prompts

1. **Be Specific:** Include concrete details about your product and goals
2. **Provide Context:** Share current performance metrics when available
3. **Set Clear Goals:** Define what success looks like
4. **Include Keywords:** List target keywords you want to rank for
5. **Share Brand Voice:** Help the AI match your brand personality
6. **Mention Constraints:** Note any limitations or requirements
7. **Ask Follow-ups:** Refine recommendations with additional questions

- **Category Optimization:** Improve entire product collections
- **A/B Testing:** Generate variations for testing
