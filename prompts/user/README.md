# User Prompts

This directory contains user-facing prompt templates and examples for the PageRankAI Shopify app.

## Files Structure

- `analysis-request.txt` - Template for requesting product page analysis
- `optimization-request.txt` - Template for requesting content optimization
- `examples.md` - Example prompts and use cases

## Purpose

These prompts help users interact effectively with the AI features of the application, ensuring they get the most valuable insights and recommendations.
